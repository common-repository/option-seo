== Description ==

With the Opti Seo you can Override titles, descriptions and keywords with page/post level input.

Some features:

a) Can Override titles, descriptions and keywords with page/post level input.
b) Auto-generate titles, descriptions and keywords when overrides are not present.
c) It helps to prevent duplicate content issues.
d) Automatically appends tags as keywords when available.
e) Makes sure your keywords are unique.



== Installation ==

1. Unzip into your `/wp-content/plugins/` directory. Make sure the files are inside the opti-seo/ folder.
2. Activate the plugin through the Wordpress 'Plugins' menu
3. Visit Settings
